# v4.8.3 Release Notes

## Fixed
- **Missing CSS on new hosts**: `entrypoint.sh` now rebuilds `styles.css` from `tailwind.css` at container startup if the compiled file is absent or older than the source. The local dev `docker-compose.yml` mounts `code/web/static/` as a volume, which overwrites the Docker-compiled CSS with the host directory — on a fresh clone the compiled file is absent (gitignored), causing broken layouts across the owned library, card browser filters, and card detail page.
