# v4.8.2 Release Notes

## Added
- **Card Detail Page** (`/cards/{name}`): Fully redesigned with a 2-column grid layout — stats table, oracle text with inline mana symbols, official rulings panel (expandable), similar cards (up to 15), price panel, and external links to Scryfall, Gatherer, and EDHREC
- **Rulings**: Cache-first lookup backed by Scryfall bulk data (`rulings_cache.json`); falls back to live Scryfall API on cache miss
- **Owned Library overhaul**: Sticky/collapsible filter panel with search, card type, color identity, theme multi-select (AND logic), CMC/Power/Toughness ranges, and card name autocomplete
- **Card Details button** on owned library tiles linking to the card detail page
- **Smart back button** on card detail page: shows "Back to Owned Library" or "Back to Card Browser" based on where you came from
- **Collapsible filter panels** on both card browser and owned library with `localStorage` persistence and active filter count badge
- **Card browser sticky filter bar**

## Changed
- Card browser filter bar stickiness moved to CSS class (no inline style)

## Fixed
- Similar cards refresh button returning 404 (route ordering issue with greedy `:path` matcher)
- LQIP blur persisting on images that were already cached before the page loaded
